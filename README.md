# ⚡ WBT DIGITAL LABS - Mega Portal Ekosistem 31 Aplikasi & Game

Selamat datang di repositori resmi **WBT Digital Labs** (WBT Technologies Corp). Repositori ini memuat ekosistem terpadu berkinerja tinggi yang terdiri dari 31 produk digital berbasis web (HTML5, Vanilla CSS, & JavaScript) yang dipisahkan ke dalam 3 divisi utama:

## 🚀 1. Utilities & Tools Offline (11 Aplikasi)
Aplikasi produktivitas ringan, super cepat, dan bekerja 100% offline di dalam browser tanpa ketergantungan server:
- **PDF Toolkit Lite**: Gabung, pisah, kompres, konversi gambar ke PDF, & proteksi password.
- **Image Compressor Pro**: Kompresi ukuran foto hingga 90% tanpa mengurangi ketajaman visual.
- **QuickScan OCR**: Ekstraksi teks otomatis dari foto dokumen atau buku.
- **Batch Image Resizer**: Ubah resolusi ratusan foto sekaligus dalam hitungan detik.
- **EXIF Photo Cleaner**: Hapus metadata lokasi GPS dan info privasi kamera.
- **Offline Markdown Editor**: Editor markdown dengan live preview & ekspor HTML.
- **SQLite Database Viewer**: Baca & jalankan query SQL pada file `.db`/`.sqlite` lokal.
- **JSON Formatter & Viewer**: Perapih struktur & pemadat kode JSON.
- **Mini API Tester**: Alternatif Postman ringan untuk pengujian REST API.
- **Bulk Photo Renamer**: Penamaan ulang massal dengan awalan & nomor urut.
- **Activity Tracker Web**: Pencatat produktivitas harian dengan visualisasi grafik.

## 🤖 2. Smart AI Suite Tools (10 Aplikasi)
Koleksi aplikasi kecerdasan buatan dengan kuota mandiri gratis harian & simulasi upgrade Lifetime PRO:
- **AI Image Captioner**: Generator caption sosmed & e-commerce.
- **AI Photo Enhancer**: Penajam foto buram & resolusi rendah menjadi HD.
- **AI Text Summarizer**: Perangkum artikel panjang menjadi poin eksekutif.
- **Background Eraser AI**: Pemotong latar belakang foto otomatis (PNG transparan).
- **Dataset Labeling Tool**: Anotasi bounding box (export YOLO `.txt` & VOC `.xml`).
- **Offline Grammar Checker**: Pemeriksa ejaan KBBI & keefektifan kalimat.
- **Offline Voice Transcriber**: Speech-to-Text real-time berbasis Web Speech API.
- **Receipt Scanner AI**: Pemindai struk belanja (OCR) ke format JSON.
- **Smart Prompt Generator**: Peracik prompt profesional Midjourney, ChatGPT, & Claude.
- **YOLO Dataset Viewer**: Visualisator koordinat bounding box dataset ML.

## 🎮 3. Arcade Game Center (10 Game HTML5)
Game HTML5 offline berkinerja tinggi untuk PC & Android lengkap dengan sistem topup resource:
- **Cube Escape Dash**: Arcade runner hindari rintangan berduri.
- **Color Block Master**: Puzzle balok grid 8x8 dengan item Palu & Sapujagat.
- **Merge Kingdom**: Idle merge pembangun kerajaan penghasil emas.
- **Mini Farming Life**: Simulasi berkebun & panen tanaman.
- **Neon Drift Racer**: Balapan mobil berkecepatan tinggi bertema cyberpunk.
- **Pixel Dungeon Quest**: RPG penjelajahan penjara bawah tanah & monster.
- **Shadow Ninja Jump**: Platformer vertikal menembus menara ninja.
- **Space Survivor**: Action shooter penghancur gelombang meteor kosmik.
- **Tower Builder 3D**: Stacking arcade pembangun menara pencakar langit.
- **Zombie Defense Idle**: Tower defense pertahanan bunker dari serangan zombie.
