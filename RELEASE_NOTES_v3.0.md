## 🔥 Peningkatan Mesin AI Unggulan (7 AI Core Upgraded)

### 1. 🖼️ CitraLabeling Studio Pro (`ai/Dataset-Labeling-Tool`)

- **Sistem Model:** Unduhan installer ringan Windows/Android tanpa bobot model berat; dukungan deteksi otomatis file `Deep Learning Neural Network TensorFlow.js + COCO-SSD.pack` seharga Rp 35.000 di folder aplikasi.

- **Mesin Segmentasi:** Integrasi **TensorFlow.js BodyPix Neural Segmentation** untuk potongan presisi helai rambut dan siluet tubuh manusia.
- **Sistem Model:** Dukungan unduhan installer ringan dan verifikasi otomatis file `BackgroundEraser_AI_Model_HD_v2.pack`.

### 3. 🎙️ Offline Voice Transcriber Studio Pro (`ai/Offline-Voice-Transcriber`)

- **Input Fleksibel:** Mendukung rekaman mikrofon langsung serta **Upload File Audio (.MP3, .WAV, .M4A)**.
- **Sistem Model:** Verifikasi otomatis file `VoiceTranscriber_Acoustic_Model_v3.pack` di folder lokal.

### 4. 🧾 Receipt & Invoice Scanner AI OCR (`ai/Receipt-Scanner-AI`)

- **Ekstraksi Otomatis:** Memindai foto struk belanja dan otomatis mengekstrak **Nama Toko, Tanggal, Item Belanja, & Total Bayar** ke format `.JSON` dan `.TXT`.
- **Sistem Model:** Verifikasi otomatis file `ReceiptScanner_OCR_Language_Model_v2.pack`.

### 5. 🎨 WBT Image & Photo Studio Pro (`tools/WBT-Image-Studio-Pro`)

- **Sistem Model:** Verifikasi otomatis file `WBTImageStudio_AI_Enhancer_Model_v4.pack`.

### 6. ⚡ AI Photo Enhancer & Upscaler Studio Pro (`ai/AI-Photo-Enhancer`)
